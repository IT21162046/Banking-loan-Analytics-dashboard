# Banking Loan Analytics Dashboard

## 📌 Project Overview

This project presents a **Banking Loan Analytics Dashboard** developed using **Power BI**. The dashboard analyzes customer loan-related data to uncover trends, customer characteristics, and lending patterns through interactive visualizations.

The objective of this project is to transform raw banking data into actionable insights that support data-driven decision-making.
